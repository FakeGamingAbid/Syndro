# Privacy Policy

**Last Updated:** March 2026

## Overview

Syndro is a privacy-first file sharing application that operates entirely on your local network. This policy explains what data we collect, how we use it, and your rights regarding your data.

## Data Collection

### What We DON'T Collect

Syndro is designed with privacy as a core principle:

- **No cloud storage** - Files never leave your local network
- **No user accounts** - No registration or login required
- **No analytics** - We don't track your usage
- **No third-party tracking** - No advertising or data brokers

### What We DO Process (Locally Only)

The following data is processed **only on your device** and is never transmitted to any external server:

1. **File Metadata** - File names, sizes, and types for transfer operations
2. **Device Information** - Device name and network address for peer discovery
3. **Transfer History** - Stored locally for your convenience (can be cleared)
4. **Encryption Keys** - Generated and stored locally for secure transfers

## Network Permissions

Syndro requires the following network permissions to function:

- **Local Network Access** - For discovering and communicating with nearby devices
- **Internet Access** - Required only for app store updates (not for file transfers)

## Data Storage

All data is stored locally on your device:
- Transfer history in local SQLite database
- Encryption keys in platform secure storage
- App settings in SharedPreferences

You can delete all local data by:
- Uninstalling the app
- Clearing app data from device settings
- Using the "Clear History" feature in the app

## Third-Party Services

Syndro does not share any data with third parties. File transfers occur directly between devices on your local network using end-to-end encryption.

## Children's Privacy

Syndro is not intended for children under 13. We do not knowingly collect personal information from children.

## Changes to This Policy

We may update this policy from time to time. Any changes will be reflected in the "Last Updated" date above.

## Contact Us

If you have questions about this privacy policy, please open an issue on our GitHub repository.
